# DTT(Digital Twin Table) 🏩
* 실제 매장의 환경을 3D 모델링을 통해 디지털 환경에서 재현한 뒤 실시간으로 매장의 사용 상황을 파악하고 예약을 진행하며 예측을 통해 디지털 트윈을 실현하는 프로젝트
